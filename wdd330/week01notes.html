LocalStorage works with any kind of data. Step 01 is to create the HTML document where the user will save their data. Then in javascript, select the place where the user puts in the data and store it in a variable.

Three functions:

1. Edit
2. Save
3. Display

The story can be edited through "localStorage. getItem (storyName)". The story is saved with the name of the author is named "localStorage. setItem (storyName, storyHTML)". Finally, to display HTML as "document. getElementById (“story_display”). innerHTML = storyHTML".
